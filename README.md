# 🌱 Мой день — ежедневный трекер роста

Личное приложение для отслеживания ежедневных задач, психологической работы над собой и духовных практик. Работает как сайт — открывается на телефоне и компьютере, устанавливается на главный экран как приложение.

---

## 📱 Как установить на телефон (Android)

1. Открой ссылку в **Chrome**: `https://andylorets.github.io/moiden-/`
2. Нажми три точки **⋮** справа сверху
3. Выбери **"Добавить на главный экран"**
4. Нажми **"Добавить"**

Готово — на главном экране появится иконка, приложение открывается без адресной строки браузера.

---

## 💻 Как открыть на компьютере

Просто открой ссылку в браузере:

```
https://andylorets.github.io/moiden-/
```

На компьютере интерфейс автоматически переключается в десктопный режим — три колонки (Утро / День / Вечер) и боковая навигация.

---

## ✨ Что умеет приложение

### 📋 Вкладка «День»
- Три фазы дня: **Утро**, **День**, **Вечер** — каждая со своим набором задач
- Чередование учёбы каждый день: **Английский → Unity → ИИ** (автоматически)
- Кнопка **?** у каждой задачи — объясняет зачем это нужно
- Отслеживание состояния утра и мотивации дня
- Проверка внутреннего конфликта бессознательного
- Поле «Что не сделал из-за страха» — вечерняя рефлексия
- Поле «Один инсайт дня»
- Возможность добавлять **свои задачи** в любую фазу
- **Стрик** — считает дни подряд когда выполнены все задачи

### 🗓 Вкладка «График»
- Стандартный распорядок дня с временными блоками
- Можно редактировать время и названия прямо в приложении
- Можно добавлять и удалять блоки

### 📖 Вкладка «Справка»
- Корень проблемы и почему это не лень
- Три страха и факты против каждого
- Техники работы с тревогой (заземление, факт vs интерпретация, протокол паники)
- Мышление которое меняет жизнь (Юнг, Франкл)

### ⚙️ Вкладка «Настройки»
- Синхронизация через **GitHub Gist** — данные доступны на всех устройствах
- Уведомления-напоминания утром и вечером
- Сброс текущего дня (стрик и смысл сохраняются)

## ☁️ Как настроить синхронизацию между телефоном и ПК

Чтобы данные были одинаковыми на всех устройствах:

1. Зайди на [github.com/settings/tokens](https://github.com/settings/tokens)
2. Нажми **"Generate new token (classic)"**
3. Название: `moiden-sync`, срок: **No expiration**, галочка: **gist**
4. Скопируй токен (начинается с `ghp_...`)
5. Открой приложение → **⚙️ Настройки** → вставь токен в поле
6. Нажми **"☁️ Сохранить в облако"**

На другом устройстве повтори шаги 5–6, но вместо «Сохранить» нажми **"📥 Загрузить из облака"**.

> ⚠️ Токен хранится только в браузере на твоём устройстве — никуда не отправляется и не виден другим.

## 🛠 Как обновить приложение

Если хочешь изменить задачи, тексты или добавить что-то новое:

1. Зайди в этот репозиторий на GitHub
2. Открой файл `index.html`
3. Нажми карандаш ✏️ (Edit)
4. Внеси изменения
5. Нажми **"Commit changes"**

Через 1–2 минуты изменения появятся на сайте. На телефоне обнови страницу или добавь `?v=2` к адресу.
